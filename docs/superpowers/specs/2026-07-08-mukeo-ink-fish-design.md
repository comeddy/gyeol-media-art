# 묵어(墨魚) Ink Fish — 발묵 대체 작품 설계

날짜: 2026-07-08 · 상태: 승인됨
배경: 1관 묵향의 발묵(潑墨)은 저해상 확산 효과가 미묘해 "반응을 느끼지 못했다"는 관람 피드백을 받았다.
수묵 테마를 유지하면서 **살아있는 생명감**을 핵심 경험으로 하는 새 작품으로 교체한다.

## 1. 컨셉

어해도(魚蟹圖) 전통의 수묵 물고기 떼가 화선지 물속을 유영한다.
1관의 다른 작품(일필=드로잉, 우중산수=풍경, 묵죽=식물)과 겹치지 않는 관 내 유일한 '동물' 소재.
발묵의 '번짐' 유산은 물고기가 남기는 먹 잔상으로 계승한다.

## 2. 시각 설계

- **물고기 12~16마리**, 시드 고정 난수(`mulberry32`)로 결정적 생성.
- 개체 = 척추 폴리라인(머리→꼬리 ~8세그먼트). 유영 파동(사인 + 위상)으로 척추가 굽는다.
  - 머리: 짙은 먹 획(둥근 타원형 스탬프)
  - 몸통: 세그먼트를 따라 굵기 감소하는 획
  - 꼬리: 갈필 2~3가닥 (일필의 브리슬 기법 재사용)
- **개체 편차**: 크기·먹 농담·유영 속도. 일부 개체는 작고 옅게(대기원근 — 묵죽 `fade()` 패턴) → 물 깊이감.
- **먹 잔상(trail)**: 오프스크린 캔버스에 몸 획을 낮은 알파로 누적, `destination-out` 페이드로 약 8초에 걸쳐 소산 (일필 strokes 버퍼 패턴 재사용).
- 한지 배경: 기존 팔레트(`#EDE6D6` 계열)·발 무늬 기법 그대로.

## 3. 행동·인터랙션 설계

기본 유영은 보이드 3규칙(분리·정렬·응집) + `noise2` 방랑. 화면 가장자리는 부드러운 반사.

| 입력 | 반응 |
|------|------|
| 없음 | 유유히 군영 (보이드 + 방랑) |
| down/느린 move | 어트랙터(먹이) — 반경 내 개체 유인, 근접 시 맴돌며 선회 |
| 빠른 move (속도 임계 초과) | 반경 내 개체 공포 — 반대 방향 대시 후 감쇠 복귀 |
| 꾹 누름 (0.4s+ 정지) | 먹방울 낙하: 그 자리에 radial 번짐 + 개체들이 모여 '마심' → 개체 먹 농담 상승(짙어짐), 이후 서서히 원상복귀 |
| up | 어트랙터 해제 → 군영 복귀 |

사운드(`audio.tone`, 스로틀): 모여들 때 낮은 오음 톤 1회, 놀라 흩어질 때 짧은 높은 톤.

## 4. 구현 구조

- **신규** `js/works/w1-mukeo.js` — 작품 모듈 계약 동일:
  `default export = { init(ctx), frame(t,dt), pointer(e), resize(w,h), dispose() }`, import 시점 부작용 없음.
- 상태: `fish[] { x, y, vx, vy, phase, size, ink, depth }`, 포인터 `{ x, y, holdT, speedSm }`.
- frame 순서: paper blit → trail 페이드(destination-out) → 물고기 잔상 stamp → trail blit → 물고기 본체 벡터 렌더.
- 성능: 16마리 × 획 ~10 ≈ 160 stroke/frame + blit 2회 — 발묵(격자 확산)보다 가벼움. 16ms 예산 내.
- selftest: 시드 고정 + 초기 배치 화면 안 분산 → 예외 0 · uniform false · <16ms 자동 충족.

## 5. 교체 범위

| 파일 | 조치 |
|------|------|
| `js/works/w1-mukeo.js` | 신규 작성 |
| `js/works/w1-balmuk.js` | 삭제 |
| `js/data.js` | balmuk 항목 → mukeo 항목 교체 (아래) |
| `making-of.html` | "한 점(발묵)은 파일럿" 서술을 "발묵 — 이후 '묵어'로 교체"로 정직하게 갱신 |
| 기타 작품 주석의 '발묵' | 기법 용어(먹 번짐)이므로 유지 |
| `docs/` 과거 스펙·플랜 | 역사 기록이므로 수정하지 않음 (본 문서가 교체를 기록) |

### data.js 항목

```js
{ id:'mukeo', wing:1, title:'묵어', en:'Ink Fish',
  desc:'먹을 머금은 물고기 떼가 화선지 물속을 유영한다. 지나간 자리마다 먹이 물에 풀린다.',
  how:'손을 대면 모여들고, 빠르게 저으면 흩어집니다. 꾹 누르면 먹방울에 모여 먹빛이 짙어집니다.', needs:[] },
```

## 6. 검증 기준

1. `node --check` 통과.
2. `?selftest&wing=1` — 1관 4작품(mukeo 포함) `ok:true, uniform:false, msPerFrame<16`.
3. 행동 검증(Playwright, 캐시 우회 fresh import):
   - 무입력 60프레임: 개체가 이동함(위치 변화) + 화면 안 유지
   - down 후: 어트랙터 반경 내 개체들의 평균 거리 감소(모여듦)
   - 빠른 move 후: 평균 거리 증가(흩어짐)
   - 꾹 누름 후: 개체 ink 농담 상승 → 렌더 픽셀 짙어짐
4. 스크린샷 육안: 붓획 물고기 형태·잔상·대기원근.
